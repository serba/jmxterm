Jmxterm

Jmxterm is an interactive command line based JMX client. It supports tab auto completion and command history browsing. Go to the wiki site for more details.

Jmxterm tracks bugs and questions in launchpad. Goto launchpad.net/jmxterm to report a bug or ask a question.
